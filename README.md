# Livescore-api
**Access and manipulate football data from [Livescore](https://livescore.com).**

## Installation

1. From pip (soon)

```sh
pip install livescore-api
```

2. From source

- Clone repo and install
```sh
git clone https://github.com/Simatwa/livescore-api.git
cd livescore-api
pip install .
```
